# Actividad Formativa 2: Simulación y análisis de señales con la Transformada de Fourier

## Descripción del proyecto
Este proyecto tiene como objetivo analizar señales en el dominio del tiempo y en el dominio de la frecuencia utilizando Python y la Transformada de Fourier.

Se desarrolló una simulación para generar y analizar señales elementales, permitiendo observar su comportamiento temporal y espectral mediante gráficas de magnitud y fase.

## Objetivos
- Representar señales en el dominio del tiempo.
- Aplicar la Transformada de Fourier usando FFT.
- Analizar magnitud y fase del espectro de frecuencia.
- Verificar propiedades de Fourier como linealidad, desplazamiento temporal y escalamiento.

## Señales analizadas
Las señales utilizadas en esta actividad fueron:

- Señal senoidal
- Pulso rectangular
- Escalón unitario

## Herramientas utilizadas
- Python
- NumPy
- SciPy
- Matplotlib
- GitHub

## Metodología
1. Generación de señales en el dominio del tiempo.
2. Aplicación de la Transformada Rápida de Fourier (FFT).
3. Obtención del espectro de frecuencia.
4. Visualización gráfica de magnitud y fase.
5. Análisis comparativo de resultados.

## Resultados
Los resultados mostraron que cada señal posee características espectrales distintas:

- La señal senoidal presenta una frecuencia dominante específica.
- El pulso rectangular genera múltiples componentes de frecuencia.
- El escalón unitario concentra energía en bajas frecuencias.

También se comprobó que cambios en el dominio del tiempo afectan directamente el espectro de frecuencia.

## Conclusión
La Transformada de Fourier permitió comprender la relación entre el dominio temporal y frecuencial de una señal. Este análisis es fundamental en áreas como telecomunicaciones, procesamiento digital de señales, audio e imágenes.
